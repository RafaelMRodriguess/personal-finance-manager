#  Gerenciador de Finanças em Python

Projeto simples para controle de receitas e despesas utilizando JSON como banco de dados.

##  Funcionalidades
- Adicionar receitas
- Adicionar despesas
- Listar dados
- Ver resumo financeiro
- Limpar dados

##  Tecnologias
- Python
- JSON



##  Estrutura

- `main.py` → código principal
- `dados.json` → armazenamento dos dados
